\# REGLAS DEL PROYECTO: FARMLOCATOR TUMBACO

1\. STACK ESTRICTO: HTML5 Semántico, CSS3 Vanilla (Flexbox/Grid), JavaScript ES6+ Vanilla.

2\. PROHIBIDO usar frameworks (React, Tailwind, Bootstrap, jQuery).

3\. MAPAS: Usar únicamente Leaflet.js vía CDN.

4\. ASINCRONÍA: Usar `async/await` y `try/catch` para consumir datos de Overpass API.

5\. ACCESIBILIDAD (WCAG 2.2): Todo HTML debe usar etiquetas semánticas y atributos ARIA (ej. `aria-hidden`, `aria-label`).

6\. IHC: Minimalismo extremo (Zero-Click Discovery). Botones interactivos deben tener un mínimo de 44x44px (Ley de Fitts).

7\. DISEÑO: Mapa renderizado en escala de grises. Solo los pines rojos tendrán color (Ley de Figura-Fondo).
